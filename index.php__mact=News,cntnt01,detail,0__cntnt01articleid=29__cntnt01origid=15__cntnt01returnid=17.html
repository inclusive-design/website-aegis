Oct 22, 2012

New version of Tecla Access for Android app is released

An update to the Tecla Access for Android app has just been released. It a number of great new accessibility features, such as:

  • A new "Android key" that provides convenient switch access to Google's Voice Search and volume controls.
  • Better error recovery from scanning selection errors in the typing keyboard.
  • A new auto-repeat lock to repeat arrow keys automatically.
  • Morse code support.

Tecla Access is available on the Google Play Store: https://play.google.com/store/apps/details?id=ca.idi.tekla&hl=en

NOTE: This app is a great keyboard for everyone, not just switch users!
